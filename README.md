# BAZAAR - интернет-магазин цветочных букетов

## О проекте

BAZAAR - это современный интернет-магазин по продаже цветочных букетов и композиций, разработанный с использованием React, TypeScript и современных веб-технологий. Проект отличается элегантным дизайном с золотыми акцентами и интуитивно понятным пользовательским интерфейсом.

## Функциональность

- Каталог букетов с фильтрацией и сортировкой
- Детальная страница товара с выбором опций
- Корзина покупок с оформлением заказа
- Личный кабинет пользователя
- Карта магазинов с использованием OpenLayers
- Адаптивный дизайн для всех устройств
- Поддержка светлой и темной темы

## Технологии

- **React** - библиотека для создания пользовательского интерфейса
- **TypeScript** - типизированный JavaScript для повышения качества кода
- **Redux Toolkit** - управление состоянием приложения
- **React Router** - маршрутизация между страницами
- **OpenLayers** - интерактивные карты
- **Webpack** - сборка проекта

## SSR, SEO и PWA в проекте

В проект добавлены следующие улучшения:

### Server-Side Rendering (SSR)

Проект использует Server-Side Rendering с помощью NestJS для:
- Предварительного рендеринга React-компонентов на сервере
- Гидратации приложения на клиенте
- Улучшения времени до первого отображения контента
- Передачи предварительно загруженного состояния Redux между сервером и клиентом

### Поисковая оптимизация (SEO)

Интегрирована SEO-оптимизация:
- Динамические метатеги для каждой страницы
- Структурированные данные Schema.org (JSON-LD)
- Карта сайта (sitemap.xml) и robots.txt
- Open Graph и Twitter Cards для социальных сетей
- Хлебные крошки для поисковых систем
- Канонические ссылки

### Progressive Web App (PWA)

Добавлена поддержка PWA:
- Service Worker для работы офлайн
- Манифест веб-приложения
- Кеширование статических ресурсов
- Оффлайн-страница
- Поддержка установки на устройство

## Начало работы

### Требования

- Node.js (версия 18 или выше)
- npm или yarn

### Установка

```bash
# Клонирование репозитория
git clone <url-репозитория>
cd bazaar-front

# Установка зависимостей
npm install
```

### Запуск в режиме разработки

```bash
npm start
```

Приложение будет доступно по адресу [http://localhost:8080](http://localhost:8080)

### Сборка для production

```bash
npm run build
```

Собранные файлы будут размещены в директории `dist/`

## Установка и запуск

### Требования
- Node.js >= 16.x
- npm >= 8.x

### Установка
```bash
# Клонирование репозитория
git clone <url-репозитория>
cd bazaar-front

# Установка зависимостей
npm install
```

### Режим разработки
```bash
# Запуск клиента и сервера в режиме разработки
npm run start:dev

# Запуск только клиента
npm run start:dev:client

# Запуск только сервера
npm run start:dev:server
```

Клиент будет доступен по адресу [http://localhost:3000](http://localhost:3000)
Сервер будет доступен по адресу [http://localhost:3000/api](http://localhost:3000/api)

### Сборка для продакшена
```bash
# Сборка клиента и сервера
npm run build

# Сборка только клиента
npm run build:client

# Сборка только сервера
npm run build:server
```

### Запуск в продакшене
```bash
npm start
```
